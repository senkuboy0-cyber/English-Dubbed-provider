# English Dubbed Provider - Cloudstream Extension

A **Cloudstream** extension for streaming **English Dubbed Anime** from **AniDb** with sub/dub support and MAL/AniList metadata sync.

---

## Features

- **English Dubbed** anime streaming
- **Sub & Dub** dual audio support
- **Multiple categories**: Top Airing, Popular, Recently Updated, Recently Aired, TV Series, Movies, ONA, OVA, Specials
- **MAL/AniList ID sync** for proper metadata
- **Trailer** support
- **Search** functionality
- **Cross-platform** support (Android, TV, etc.)

---

## Supported Source

| Source | Language | Type | URL |
|--------|----------|------|-----|
| [AniDb](https://anidb.app) | English | Anime, AnimeMovie, OVA | https://anidb.app |

---

## Installation

1. Download the latest `.cs3` or extension file from [Releases](https://github.com/senkuboy0-cyber/English-Dubbed-provider/releases)
2. Open **Cloudstream** app
3. Go to **Settings** -> **Extensions**
4. Tap on the **+** button and select the downloaded file
5. The extension will be installed automatically
